#  格式说明符简介，适用于printf()和scanf()函数

|  名称 |  作用 |
| ------------ | ------------ |
|sd|以带符号的十进制形式输入输出整数|
|So|以无符号的八进制形式输入输出整数|
|sx|以无符号的十六进制形式输入输出整数|
|su|以无符号的十进制形式输入输出整数|
|sc|以字符形式输入输出单个字符|
|ss|输入输出字符串|
|sf|以小数点形式输入输出单、双精度实数|
|se|以标准指数形式输入输出单、双精度实数|
|sg|选用输入输出宽度较小的格式输出实数|
